Open Trivia Database

Open Trivia Database (OpenTDB) provides a substantial amount of questions for Smarter. It contains more than 4000 verified questions of varying difficulties and categories. Smarter uses an API called Otriviata that sources its questions from Open Trivia Database but also adds an ID to each question. This makes it possible for you, the user, to take note of some question ID you see while browsing and add it to your game afterwards. It is not possible to contribute to the database directly through Smarter, although you are welcome to do so on their website.

All Open Trivia DB questions are licensed under CC BY-SA 4.0
